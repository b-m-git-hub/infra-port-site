## Features

* **Terraform**
    * **Infrastructure Provisioning:** Define cloud resources, manage DigitalOcean droplets, and configure firewall and DNS.
* **Ansible**
    * **Configuration Management:** Nginx, automated SSL/TLS certification renewal via Certbot, and security headers.
